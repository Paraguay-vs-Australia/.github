## Тактика и расклад

**Парагвай** будет давить с первых минут — без ничьей они скорее выбывают. Но **без Альмирона** атака очень ограничена. Против Турции Парагвай нанёс мало ударов, но удержал счёт и в меньшинстве.

**Австралия** может позволить себе сыграть закрыто: ничья = выход в плей-офф. Схема 5-4-1 — привычная для Поповича оборона с перестроением. Иранкунда — быстрый контратакующий удар.
